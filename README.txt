Lunite Tunnel Is Multi Sms Gateway Bridge For WooCommerce Wordpress Sites.You can change to any preferred SMS gateway service anytime.

== Description ==

Lunite main goal is to help the non tech peoples support the any sms gateway as they needed.Currently, we are supporting
* [Ada Reach Sms Gateway](https://ada-asia.com/reach/)


== Frequently Asked Questions ==
= How I add my own Sms gateway =
You have to contact the developer in order to support new sms gateway.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Download the latest release
2. Login into Wordpress plugin Area
3. Install the Zip File
4. In the Admin Section You will see the Lunite Section.
5. Go there and Select your Sms Service that your going enable for woocommerce site.
6. Add the credentials and press Save.
7. Try to do sample Order . Within Just seconds you will receive the sms.
